# Journaller

A Simple, HTML Based, Journal Entry Generator/Editor - designed to be fully featured in a single page.

## Features
- Date/Titles
- Various Export Options
- Dark/Light Mode

### Export Options
- CSV
- JSON
- Markdown
- Clipboard (In Markdown)
